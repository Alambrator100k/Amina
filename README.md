# موقع تهنئة لأمينة بمناسبة حفظ القرآن الكريم في شهر رمضان المبارك

![شهر رمضان المبارك](https://img.icons8.com/color/96/000000/quran.png)

## 🌙 نظرة عامة

موقع ويب تفاعلي وجميل جداً لتهنئة **أمينة** بمناسبة حفظها للقرآن الكريم في شهر رمضان المبارك. تم تصميم الموقع بأسلوب إسلامي راقي يجمع بين الجمال البصري والروحانية.

## ✨ المميزات

- **تصميم جذاب** بألوان إسلامية (أخضر، ذهبي، نيلي)
- **تأثيرات بصرية مذهلة**:
  - نجوم متلألئة في الخلفية
  - زخارف قرآنية متحركة
  - تأثيرات ظهور تدريجية للعناصر
  - بطاقات تفاعلية مع تأثير التحويم

- **محتوى مخصص للمناسبة**:
  - رسالة تهنئة مؤثرة
  - آيات قرآنية كريمة
  - فضائل حفظ القرآن
  - كلمة ختامية مؤثرة

- **ميزات تفاعلية**:
  - مؤثرات صوتية (أذان) مع زر تحكم
  - تنبيه ترحيبي تلقائي
  - تأثير كتابة النص الرئيسي
  - متجاوب مع جميع أحجام الشاشات

## 🛠 التقنيات المستخدمة

- **HTML5** - هيكل الموقع
- **CSS3** - التصميم والتأثيرات البصرية
  - Flexbox و Grid
  - Keyframes animations
  - Media queries للتجاوب
- **JavaScript** - التفاعلية والمؤثرات
  - DOM manipulation
  - Audio control
  - Scroll effects
